# Consumer Web Service
### Consumir Queue Objeto:


**WS:** *\consumerPerson*

**Exemplo:** ```http://localhost:9500/consumerPerson```


### Consumir Queue Texto:

**WS:** *\consumerMessage*

**Exemplo:** ```http://localhost:9500/consumerMessage```

### Consumir Queue Route File:

**WS:** *\startRouteConsumer*

**Caminho:** ```C:\routeOut```
